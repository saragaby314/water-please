# Water Please

## 1. Descripción del proyecto
Water Please es una aplicación web diseñada para ayudarte a mantener un hábito saludable: **beber suficiente agua todos los días**.  
El proyecto surgió al darme cuenta de que muchas veces olvidamos hidratarnos, incluso cuando es muy importante para nuestra salud, energía y concentración.

---

## 2. Problema que resuelve
Olvidar beber agua durante el día. Esta app permite:  
- Registrar tus datos personales y físicos (edad, peso, altura, sexo).  
- Establecer tu meta diaria de agua según actividad física y clima.  
- Llevar un **registro del agua consumida** durante el día.  
- Dar consejos y tips de hidratación según la estación y temperatura.

---

## 3. Usuario principal
Principalmente es para uso personal, pero puede ser útil para cualquier persona interesada en mantener un hábito saludable de hidratación.

---

## 4. Funcionalidades Core (MVP)
- Introducir datos personales en un formulario.  
- Registrar el progreso diario de agua consumida.  
- Mostrar visualmente el progreso con barras de progreso.  
- Sección de consejos y tips saludables.

---

## 5. Funcionalidades adicionales (Nice-to-have)
- Alertas según la temperatura y estación del año para beber más agua.  
- Blog personal con experiencias de hidratación.  
- Adaptación de metas según actividad física.  

---

## 6. Tecnologías utilizadas
- HTML5  
- CSS3  
- Diseño responsive 

---

## 7. Estructura de archivos
Water-Please/ ← Carpeta raíz del proyecto
├─ index.html ← Página principal (progreso diario)
├─ progreso.html ← Configuración de perfil y registro
├─ consejos.html ← Tips y consejos de hidratación
├─ blog.html ← Blog y experiencias personales
├─ style.css ← Estilos globales
└─ README.md ← Descripción del proyecto

---

## 8. Cómo ejecutar el proyecto
1. Clona o descarga el repositorio.  
2. Abre cualquier archivo `.html` en tu navegador.  
3. No se necesita servidor ni base de datos; todo funciona **localmente**.
